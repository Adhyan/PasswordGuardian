    Password Security 101

    Understand the attacks your password defends against

    🎲

    What is Entropy?

    Entropy measures unpredictability in bits. A password with 60 bits of entropy means an attacker needs to try up to 260 ≈ 1 quintillion combinations. Each additional character from a large pool roughly doubles the search space.

    H = L × log₂(R)

    L = length  ·  R = character pool size

    💥

    Brute Force Attack

    An attacker systematically tries every possible combination until the correct password is found. Modern GPUs can attempt billions of MD5 hashes per second. A 6-character lowercase password falls in milliseconds. Length is your best defence.

    📖

    Dictionary Attack

    Instead of random guessing, attackers start with known words, names, and phrases, then apply common transformations: password → P@ssw0rd. Any recognisable word in your password is a weakness, regardless of substitutions.

    🌈

    Rainbow Tables

    Pre-computed tables that map hash values back to passwords. An attacker who steals an MD5 hash database can look up most passwords instantly. This is why modern systems use salted hashing (bcrypt, Argon2) — unique salts make rainbow tables useless.

    🔄

    Credential Stuffing

    Billions of username/password pairs from past breaches are freely available. Attackers automate login attempts across thousands of sites. If you reuse a password from any breached service, every account sharing that password is at risk.

    ♻️

    Password Reuse

    Reusing passwords is the single most dangerous habit. One breach exposes every account. Use a password manager to maintain unique, high-entropy passwords for every service — you only need to remember one master password.

    ✅

    Best Practices

    • Use a minimum of 16 characters for important accounts.
    • Combine uppercase, lowercase, digits, and symbols.
    • Never use dictionary words without heavy obfuscation.
    • Avoid keyboard walks: qwerty, asdf, 1234.
    • Use a passphrase (four random words) for memorable strong passwords.
    • Enable multi-factor authentication wherever available.
    • Use a password manager: Bitwarden, 1Password, or KeePass.
    • Check breaches at HaveIBeenPwned.
